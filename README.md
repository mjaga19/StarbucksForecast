# StarbucksForecast
Objective:
To analyse the Starbucks company sales revenue dashboards for the FY 2018 -2024, and for Time series and Benchmarking dashboards, list my observations and reasons for the same. 


overview
•5MA symmetrical smoothing technique is used for initial forecast.
•All three segment forecast charts are plotted and included for post-pandemic period
oLOB – Beverages, Food, Other(LOB)
oMARKET – North- America, International, Other(MARKET)
oSTORE TYPE – Company-Op, Licensed, Other(Store)
•YoY% growth change is calculated for each listed item in the segments.
•Observations and recommendations are included.
