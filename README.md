# StarbucksForecast
Objective:

To analyse the Starbucks company sales revenue dashboards for the FY 2018 -2024 and provide sales forecast for Year 2025


Overview
1.	5MA symmetrical smoothing technique is used for initial forecast.
2.	All three segment forecast charts are plotted and included for post-pandemic period
      * LOB – Beverages, Food, Other(LOB)  	
      * MARKET – North- America, International, Other(MARKET)
      * STORE TYPE – Company-Op, Licensed, Other(Store)
       
4.	YoY% growth change is calculated for each listed item in the segments.
5.	Observations and recommendations are included.
